
##Get Loco API Documentation

*Note:: This is a test application and has no real-world use.*


***Get All Locations***

```GET /api/v1/locations```

***Get Location by ID***

```GET /api/v1/locations/{id}```

***Store Location*** 

```POST /api/v1/location```

***Update Location***

```PUT /api/v1/location/{id}```

***Delete Location***

```DELETE /api/v1/location/{id}```
